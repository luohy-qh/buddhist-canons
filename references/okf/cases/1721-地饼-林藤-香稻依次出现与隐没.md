# 地饼、林藤、香稻依次出现与隐没

地味隐没后大地生地饼，味殊妙馨香甘美；众生贪著行不善，地饼隐没；复生林藤，色味甘美；又贪著行不善，林藤隐没；后生自然香稻，依时成熟，食后腹内生碍，遂生男女根。
