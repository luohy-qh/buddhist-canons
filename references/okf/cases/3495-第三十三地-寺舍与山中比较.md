# 第三十三地：寺舍与山中比较

入寺舍及在俗人家作坛，百倍千倍不及山中。欲救众生终须择地。寺上伽蓝穿地深二尺，内得骨者必不堪，得瓦石者仍埋七宝于地内始得用。
