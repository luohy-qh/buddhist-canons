# 昙始化辽东及护法

释昙始，关中白足和尚，晋大元末年赉持经律数十部往化辽东。后关中遇害刀不能伤，拓跋焘毁灭佛法时闭绝隐匿。太平末算知焘化将至，入宫示现神异，使焘悔过，光复竺教。
