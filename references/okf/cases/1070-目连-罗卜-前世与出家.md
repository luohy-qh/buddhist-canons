# 目连（罗卜）前世与出家

目连在俗时名罗卜，深信三宝，欲往他国兴易，支分财宝令母设斋供佛僧，母生悭吝隐匿财资，欺诳凡圣，命终堕阿鼻地狱。罗卜三周礼毕投佛出家，证阿罗汉果。
