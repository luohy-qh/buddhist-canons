# 婆薮槃豆造俱舍论

婆薮槃豆后造七十真实论破僧佉论。又讲毗婆沙义，每日造一偈摄义，刻赤铜叶置醉象头下宣令，无人能破，共造六百余偈（俱舍论偈）。后应罽宾师请作长行解偈，名阿毗达磨俱舍论。
