# 买乳得酪喻

人持瓶买乳湩，寄放牧牛家，后取时湩已转成酪。买家争辩，牧牛家称是汝故乳自转成酪。王判牧牛家直。喻今世用名身作善恶，乃生后世，今世作善恶是其本。
