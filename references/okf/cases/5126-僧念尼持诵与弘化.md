# 僧念尼持诵与弘化

僧念本姓羊，泰山南城人。十岁出家，为法护尼弟子。贞节苦心，禅思精密，诵法华经日夜七遍。宋文孝武二帝常加资给。齐永明中移住禅林寺，禅范大隆，诸学者众。司徒竟陵王四事供养。年九十，梁天监三年卒。
