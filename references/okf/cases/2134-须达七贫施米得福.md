# 须达七贫施米得福

须达最为贫困时，从粪壤中得一栴檀木斗，卖得四斗米。先后遇舍利弗、目连、迦叶、佛来乞食，其妇将全部米布施。佛咒愿后，家中珍宝谷帛自然充满，须达复富。
