# 然灯佛授记释迦牟尼成佛

佛叙述值遇胜观如来时未被授记，后经阿僧祇劫，然灯如来出现于世，佛时为胜云，持优钵罗华七茎供养，得授记未来成佛名释迦牟尼，并证得无生法忍。
