# 十二缘生支的详细说明

愚闇名无明，造作名为行，了别名为识，互相摄持名为名色，依处所故名为六处，触境故名为触，领纳故名受，渴爱故名为爱，取著故名为取，取复生有故名为有，能生故名生，根熟故名老，灭坏故名死。
