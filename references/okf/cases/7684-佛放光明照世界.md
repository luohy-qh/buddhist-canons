# 佛放光明照世界

佛从足心放六万亿百千光明，十足指放十亿百千光明，两胁、两膝、两脚、两肩、两肘、脐、头、两手指、两臂、两眉、项、两眼、两耳、鼻、四面、四十齿、眉间相、顶结相各放不同数量光明，照此三千大千世界无所不周，普曜十方江河沙等诸佛国土。
