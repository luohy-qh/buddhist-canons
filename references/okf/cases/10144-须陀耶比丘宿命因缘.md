# 须陀耶比丘宿命因缘

须陀耶比丘过去世为小家子，值遇众僧大会讲法，闻法欢喜，以一瓶酪布施众僧，因此福报寿终生天，下生世间财富无限，九十一劫豪尊荣贵。
