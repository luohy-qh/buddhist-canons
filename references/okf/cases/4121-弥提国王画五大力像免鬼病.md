# 弥提国王画五大力像免鬼病

佛灭后千三百年，弥提国百鬼乱入，疾疫流行。王臣依智臣建议，令画五大力菩萨像并设仁王斋会，菩萨放光，诸恶鬼神出离国土，病者消除，国内安稳。此后每年初月八日设斋，相续三百年不绝。
