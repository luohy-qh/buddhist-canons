# 张无尽参兜率悦禅师悟道

张无尽因对香严独脚颂等有疑，被兜率悦禅师点破，后触翻蹋床忽然省悟，作颂呈悦。悦认可其悟，但指出尚有极微细处未破，并作颂证之。
