# 慧思和尚三生修行与方广寺缘起

陈高僧思大和尚（惠思）在衡山修行，曾于梧桐坡石冈上告知弟子自己已三生在此修习，掘地得骨，并建般若台居之。居十年后圆寂，后转世为圆泽，与李源有“三生石”之约。
