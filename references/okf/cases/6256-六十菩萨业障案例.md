# 六十菩萨业障案例

过去于拘留孙如来教中出家，因诽谤、嫉妒、破坏二法师及其信众，导致堕入阿鼻、活、黑绳、热地狱，后得人身五百世生盲，再于正法灭时生于恶国贫贱处，最终业障灭尽得生阿弥陀国。
