# 太子无念得经奉行七千岁

过去久远世，遮迦越王之子无念，年十六时闻宝具足有德行王如来说陀邻尼经，得已讽学常念奉持，精进不睡胁不在席七千岁，不念爱欲、财宝、他事，常独一处意不倾动七千岁，后闻九十九亿不可计佛说智慧法，弃国捐王行作沙门九万九千岁。
