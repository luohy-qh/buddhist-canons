# 青颈观自在菩萨画像法

其像三面：正面慈悲凞怡，右边师子面，左边猪面。首戴宝冠，冠中有化无量寿佛。四臂：右第一执杖，第二执莲花；左第一执轮，左第二执螺。以虎皮为裙，黑鹿皮于左膊角络，被黑蛇为神线，于八叶莲花上立。
