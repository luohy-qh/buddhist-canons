# 文殊师利三月不现而教化众人

文殊师利在夏三月不现于佛边、众僧、请会、说戒中，而是在舍卫城和悦王宫采女、淫女、小儿中度过，开解五百女人，教化和悦王宫中采女，令五百童子及五百童女立不退转。
