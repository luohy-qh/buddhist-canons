# 四梵志贡石蜜得度

四位有神通的婆罗门各执甘美石蜜，依次至如来所奉上。如来对第一人说“所行非常”，第二人说“谓兴衰法”，第三人说“夫生辄死”，第四人说“此灭为乐”。四人初掩耳不承受，后各自说偈即心开意解得阿那含道，后至佛所出家成罗汉。
