# 四方礼之具体仪轨

初礼东方阿閦佛，全身委地以心礼，金刚合掌舒顶上；次礼南方宝生佛，金刚合掌下当心，以额著地；次礼西方无量寿佛，金刚合掌置顶上，以口著地；后礼北方不空成就，金刚合掌当心上，用顶著地。
