# 偷盗业果报于地狱

罪人侵取他财摄为己有，狱卒执取后持利刀割截支节断坏，断已还合；或持利剑、烁吉帝、都摩啰、捣杵等皆有炽焰，斩斫打击种种治罚。经无数百千岁中在是地狱，乃至偷盗业果坏散极尽边际然后放释。
