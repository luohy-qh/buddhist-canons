# 十斋日持斋念佛灭罪

十斋日各日念特定佛号可除斋除罪并免堕特定地狱：一日念定光如来除罪四十劫免刀枪地狱；八日念药师琉璃光佛除罪三十劫免粉草地狱；十四日念贤劫千佛除罪一千劫免镬汤地狱；十五日念阿弥陀佛除罪二百劫免寒冰地狱；十八日念观世音菩萨除罪九十劫免剑树地狱；二十三日念卢舍那佛除罪一千劫免饿鬼地狱；二十四日念地藏菩萨除罪九十劫免才截地狱；二十八日念阿弥陀佛除罪一千劫免铁锯地狱；二十九日念药王药上菩萨除罪七千劫免硙磨地狱；三十日念释迦牟尼佛除罪八千劫免寒冰地狱。
