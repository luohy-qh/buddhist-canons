# 弥勒降生与出家因缘（贤愚经）

弥勒生于波罗捺国辅相之家，初生有三十二相，因母怀胎后性情变慈而得弥勒名。后因国王畏惧，父母送其随舅波婆利学习，后遣往佛所，七岁发心，后从佛出家。
