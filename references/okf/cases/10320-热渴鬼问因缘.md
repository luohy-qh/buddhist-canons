# 热渴鬼问因缘

一鬼问恒患热渴行见恒河冀入其中以除热渴方入其中身体焦烂肌肉离骨渴欲饮人一口之腹五藏焦烂痛不可言罪因，目连答因前世喜焚烧山泽残害众生，今受花报果入地狱。
