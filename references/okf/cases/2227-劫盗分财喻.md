# 劫盗分财喻

一群盗贼分赃，劣等布料分给最劣者，此人愤怒，但后来卖得高价，一人所得倍于众伴，方生欢喜。比喻世人不知布施有报，少施得生天上，受无量乐，方悔不广施。
