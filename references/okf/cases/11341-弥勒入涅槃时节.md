# 弥勒入涅槃时节

据弥勒观经，弥勒于却后十二年二月十五日，在波罗捺国劫波利大婆罗门家本所生处结跏趺坐入灭尽定取灭度，后生兜率天，五十六亿七千万岁后下生作佛。
