# 德山见沩山公案

德山到沩山法堂，从东过西从西过东，顾视云‘无无’便出；后复入相见，提起坐具唤‘和尚’，沩山拟取拂子，德山便喝拂袖而出；沩山晚问首座，首座答已去，沩山评此子日后将呵佛骂祖。
