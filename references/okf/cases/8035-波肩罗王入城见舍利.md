# 波肩罗王入城见舍利

波肩罗王到达拘尸城，见城门有咒师防止外难，咒师告知佛涅槃已过四七（二十八日），大众正分舍利。王入城后见七宝庄严师子座，安置七宝舍利金坛，大众悲哀供养。
