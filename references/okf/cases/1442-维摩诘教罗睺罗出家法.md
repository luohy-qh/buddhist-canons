# 维摩诘教罗睺罗出家法

维摩诘先呵罗睺罗不应说功德利，后广明教相，分自利（离三相、离六十二见、处涅槃等五句）与利他（降魔、度五道、净五眼、得五力立五根等十八句），并教诸长者子出家仪式。
