# 陀罗尼自在王菩萨于地狱镬缘说法救苦

惠生禅师被请入阎魔王宫，见地狱中有无量沙门受苦。陀罗尼自在王菩萨于铁镬缘上为罪人说法，令忆念昔罪、愧忏厌离。惠生因见闻此事而悲喜交集。
