# 襄州岘山华严寺行像鼻洟出兆

周襄州岘山华严寺有一木像，高五丈许，隋文将崩时，像两鼻洟出沾污怀中，拭之无尘；贞观二十三年四月又出，后太宗升遐；同年六月再出，七月汉水泛涨成灾。
