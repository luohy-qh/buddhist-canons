# 牸牛一日杀三人，源于前世贾客欺老母

瓶沙王问佛一头牸牛为何杀三人，佛说往昔三贾客欺老母舍直，老母咒愿后世相逢必杀，今牸牛即老母，三贾客即被抵死者。
