# 龙树菩萨助扬佛法

龙树菩萨自言于往昔燃灯佛时，在五浊恶世作白衣教化众生，国王拟杀害之，彼时化现大神变令国王心生恭敬共兴佛教。今得闻如来此法，我今助之。
