# 多罗菩萨现身应愿之瑞相

多罗菩萨即现其身告行者言：汝求何愿一切施与。纵修行者有积业重障，亦现警诫。或放光明或闻雷震钟磬等声，或于空中无云而雨，香华清凉妙风触行者身。
