# 造塔供佛福胜七宝布施四大洲

以七宝作四大洲布施众僧，不如如来般涅盘后以泥团作窣堵波如阿摩勒果、上安相轮如针、伞盖如枣叶、中安佛像如麦粒、下葬舍利如白芥子，此福更胜。
