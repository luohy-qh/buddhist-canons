# 大像西畔两铺坐像

大像西畔两铺坐像，一是玛瑙，用金银玉三种镂之；一是白银，以黄金镂之。二像并用七宝填之，各坐宝莲花亦高丈八，是金刚幢菩萨于维式佛时造之。释迦如来成道经十五年，问此像过去诸佛说何经，此像答云过去有十三亿佛，说毗尼教一一不同，三藏正典在娑竭龙宫，阴阳书数在须弥山金刚窟中。
