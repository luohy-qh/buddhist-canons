# 食果之过与制戒

影胜王将千树果林施与僧众，六众苾刍在果小时即食尽，致使他国使者求果不得，摩揭陀国人集会时亦无果可食，众人讥嫌沙门不知足。佛因此制戒：不应食果，食者得越法罪。
