# 冯尼神通与预言

冯尼本姓冯，高昌人。年三十出家，菜蔬一食戒行精苦，烧六指供养皆悉至掌。诵大般涅槃经三日一遍。曾预言法惠法师往龟兹国金花帐下直月处得胜法，后果应验。高昌诸尼莫不师奉。年九十六卒。
