# 除障佛顶轮王现三摩地之瑞应

世尊入除障三摩地，从顶上发生惹耶三摩地，状若轮王之像，白色首戴五佛宝冠，手执金刚钩，项背圆光，通身如车轮状，晖曜赫奕。现此三摩地时，十方世界六种震动，一切地狱六趣众生应堕恶道者皆悉灭除一切恶业不复受，便得生天及十方清净国土。善住天子七返恶道之身一时消灭，是故号为除障佛顶轮王。
