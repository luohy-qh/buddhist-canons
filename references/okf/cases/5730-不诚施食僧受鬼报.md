# 不诚施食僧受鬼报

山中僧病笃见鬼因法师不诚欲拽下地；另有一僧不诚被鬼舁至河中欲沈，一僧心存匙钥致鬼见饭上铁片，一僧念衣致鬼见饭上兽毛，皆受显报。
