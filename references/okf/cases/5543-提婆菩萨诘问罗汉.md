# 提婆菩萨诘问罗汉

提婆菩萨在城西故寺诘问温呾罗罗汉，七转后罗汉无法回答，私运通力往慈氏处请教，得知提婆是贤劫之佛，弹指间返回继续对谈，菩萨知是慈氏所释。
