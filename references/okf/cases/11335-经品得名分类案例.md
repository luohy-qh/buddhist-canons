# 经品得名分类案例

经有二十八品，总为四例：一义为名有十五（如从法、从喻、从人、从事），二义为名有十（如人法、能所、因果），三义为名有一（五百弟子授记品），不定为名有二（安乐行品、见宝塔品）。
