# 太子无念奉持经法得果

过去久远劫，太子无念闻佛说陀邻尼经，欢喜得经，讽学常念奉持，精进不睡胁不在席七千岁，不念爱欲、财宝、他事，独处意不倾动七千岁，后弃国捐王行沙门，九万九千岁奉行陀邻尼，教化八十亿万那术人发无上道意。
