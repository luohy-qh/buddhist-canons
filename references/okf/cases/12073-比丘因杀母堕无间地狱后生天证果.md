# 比丘因杀母堕无间地狱后生天证果

一青年因贪欲杀母，随后出家修行精进，成为寺主讲经说法，死后却因杀母罪堕入无间地狱。经弟子罗汉救助，凭佛法威德生夜摩天，后见佛闻四谛法证预流果。
