# 六群比丘嗔嫉言比丘为利养故教授比丘尼

六群比丘自知不复得教戒尼。嗔嫉心故便作是言。诸比丘为衣食时种种利养为尼说法。因是制不听作如是言。随其所说一一物中一一犯堕。
