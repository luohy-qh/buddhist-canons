# 毗世师论师：微尘是常能生一切物是涅槃因

第七外道毗世师论师说地水火风虚空微尘物功德业胜等十种法常，和合而生一切世间知无知物，无和合者即是离散，散者即是涅槃，微尘是常能生一切物是涅槃因。
