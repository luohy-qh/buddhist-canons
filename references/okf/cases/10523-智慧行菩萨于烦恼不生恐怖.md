# 智慧行菩萨于烦恼不生恐怖

以猛火不怖草木、日轮不与闇住、大风不障、虚空劫火不烧、宝铁爱不住不净、虚空不与地合、铁围山风不能动、仓鹄取乳不取水等比喻，说明智慧行菩萨于诸烦恼不生恐怖、不与惑住、不被障碍、不为所动、只取智不取烦恼。
