# 奇特众

如佛高台楼观师子之座。诸庄严内。一一各出一佛世界尘数菩萨。谓海慧等。此是如来依报所摄。以表依正无碍故。人法无二故。又如佛眉间出胜音等佛世界尘数菩萨。以表因果无碍故。又表从证起教故。
