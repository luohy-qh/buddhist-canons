# 欲邪行业果报于灰河地狱

染欲者见女人在灰河地狱叫呼怖畏，闻其号哭声业幻痴迷，即时奔入河中，流焰触身遍体融溃，业力所持复还活；又见女人叫呼求救，奔前持捉，女人返捉前人，以炽焰铁棒打击，血流遍满全身碎坏，业力所持即时还活，染因重故又复奔前，同在灰河大地狱中经无数百千之岁。
