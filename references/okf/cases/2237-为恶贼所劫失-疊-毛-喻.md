# 为恶贼所劫失[疊*毛]喻

二人同行，一人被贼剥衣，衣头裹金钱，欲赎衣反示贼金钱，贼取衣及金而去。比喻修行者为烦恼贼劫掠，失善法丧功德，且使他人失道业。
