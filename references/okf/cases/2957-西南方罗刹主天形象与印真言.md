# 西南方罗刹主天形象与印真言

罗刹主天乘白师子，身著甲胄，右手持刀令竖，左手大指押中小二指，赤肉色。二天女侍左右，二罗刹鬼持三股戟。印相：右手作拳安腰右，左手五指竖相著屈地水中节，以空端押地水甲。真言：南莫三满多没驮南乃哩底曳(二合)娑嚩(二合)贺。
