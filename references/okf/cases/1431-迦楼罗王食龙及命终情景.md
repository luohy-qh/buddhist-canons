# 迦楼罗王食龙及命终情景

化生迦楼罗王名正音，寿八千岁，日食五百小龙。命终时诸龙吐毒，饥火所烧，七返上下后至金刚轮山顶命终，身肉消散，唯心留存如人膝大。
