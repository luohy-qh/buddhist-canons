# 四土对应四因

一、菩萨修四种愿行之因，众生未断界内缘集，生同居土；二、权实四行，众生断界内结尽，生有余土；三、别圆教愿行，众生断无明，生果报土；四、圆断愿行，因极果满，居常寂光。
