# 东方帝释天形象与印真言

东方帝释乘白象王，住五色云中，身金色，右手持三股当心，左手托左胯，左脚垂下。三天女各手持莲花。印相：右手作拳安腰，左手五指直竖相著，地水二指屈中节，以风著火背，空屈中节。真言：曩莫三曼多没驮南因捺罗耶娑嚩(二合)贺。
