# 七佛侍者弟子

毗婆尸佛侍者阿输迦；尸弃佛侍者刹摩迦噜；毗舍浮佛侍者乌波扇睹；俱留孙佛侍者没提逾；俱那含牟尼佛侍者稣噜帝里野；迦叶佛侍者萨里嚩蜜怛啰；释迦牟尼佛侍者阿难陀。
