# 生忍因缘十事

生忍因缘有十事：一不观于我及我所相，二不念种姓，三破除憍慢，四恶来不报，五观无常想，六修于慈悲，七心不放逸，八舍于饥渴苦乐等事，九断除嗔恚，十修习智慧。
