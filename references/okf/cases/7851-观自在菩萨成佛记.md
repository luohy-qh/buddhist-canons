# 观自在菩萨成佛记

无量光如来涅槃后，观自在菩萨在众宝庄严菩提树下成等正觉，号普明高显吉祥峰王如来。其佛刹功德庄严远超师子游戏金光王如来刹土，且纯一清净大菩萨众，无有声闻缘觉名字。
