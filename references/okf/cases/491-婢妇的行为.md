# 婢妇的行为

婢妇心常畏忌不敢自慢忠孝尽节，口不粗言身不放逸，以礼自防如民奉王，夫婿敬幸不得憍慢，若得杖捶敬承奉受，及见骂辱默然无辞，甘身苦乐无有二心，募修妇道不择衣食，事夫如事大家。
