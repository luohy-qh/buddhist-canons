# 如来灭后发菩提心以香华入寺称名之利益

有人于千万岁亲近佛备修供养，另有人于佛灭后发菩提心以一香华入寺，举足下足称南无世尊，以此深解殷重信心供养舍利塔，不以求报不疑惑，中间若千万亿劫不堕恶道。
