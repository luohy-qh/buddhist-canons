# 下品中生造恶与往生

毁犯五戒八戒及具足戒，偷僧祇物、盗现前僧物，不净说法，无有惭愧，应堕地狱，临终遇善知识赞说佛十力、光明、五分法身，除八十劫生死之罪，地狱猛火化为清凉风，往生七宝池莲华内，经六劫花开，闻大乘甚深经典，发无上道心。
