# 善胜王施身饲蚊鹫

过去善声国善胜王，以广大施周给人民乃至细虫，脱衣饲蚊虻；帝释化鹫欲啄其目，王无惊慑，愿以身肉恣食；帝释化婆罗门索眼，王亦无吝。帝释现形赞叹，谓王当证菩提。
