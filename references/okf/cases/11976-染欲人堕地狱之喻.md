# 染欲人堕地狱之喻

染欲人如堕秽污井者，自染欲泥而复教他人亦染；如获怨对弃掷火坑，自堕欲火坑复教他人亦堕；如教人登刀锋山受快乐，世间父母爱恋子息亦复如是，以爱子故于欲染事随生取著，而欲染法实大恶如毒蛇，作是说者当堕地狱饿鬼畜生道中。
