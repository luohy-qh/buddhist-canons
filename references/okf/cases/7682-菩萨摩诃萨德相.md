# 菩萨摩诃萨德相

菩萨摩诃萨得诸总持逮成三昧，修于空行尊于无想，不念众愿以得等忍，制览无数皆得五通，所说经法不侥供养，致深妙法度于无极，得无所畏超越魔事，脱于一切阴盖之碍。
