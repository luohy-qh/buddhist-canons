# 错误事佛者的结果

善神离之，恶鬼得其便，因衰病之，疾病，不能自信，求医卜问无益，祷赛邪神，众过遂增，妖魅恶鬼屯守其门，丧衰死亡，财产衰耗，家室病疾相续，命终堕泥犁中。
