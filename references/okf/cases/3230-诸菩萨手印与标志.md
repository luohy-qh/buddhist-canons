# 诸菩萨手印与标志

施无畏菩萨作施无畏手；除一切恶趣定慧舒合掌；救护慧菩萨悲手掌在心，直竖空向上；大慈生菩萨慧风空持华，悲旋润右置；除一切热恼作垂施愿手，甘露水流注指端；不思议慧以无畏之手空风持珠状。
