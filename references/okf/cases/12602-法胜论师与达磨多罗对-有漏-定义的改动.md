# 法胜论师与达磨多罗对‘有漏’定义的改动

佛灭后五百年法胜论师造阿毗昙心论说‘若生烦恼是圣说有漏’，六百年达磨多罗造杂阿毗昙心论改‘生’为‘增’，即‘若增诸烦恼是圣说有漏’。
