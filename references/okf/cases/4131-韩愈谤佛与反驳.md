# 韩愈谤佛与反驳

韩愈原道佛骨如同傅奕章疏。其说佛为夷狄之法、有佛则国祚短、释教赘疣等，被反驳：佛生于天竺为南阎浮提之中；寿夭系于三世因果；释教与儒教同劝善。
