# 受请应供法示例

长者请僧，佛言：若正趣正向人皆已礼请；虽非正趣正向，若为解脱出家、坐禅人、读诵经人、劝作众事人，除恶戒人，余一切僧皆应食。若请僧应二众食（比丘及沙弥），若请二部应五众食。
