# 比罗陀守父嘱供僧得福

长者修陀罗嘱子比罗陀每年腊月八日供佛，后家贫，比罗陀借钱供办，佛与僧食毕，夜半故藏自然盈满，夫妻问佛，佛说七财具足所致。
