# 饿鬼比丘因触众僧净食而堕饿鬼

新得阿罗汉的比丘在厕边见到饿鬼比丘，饿鬼比丘自述因曾以不净食食众僧，故五百余年不见浆水，靠阿罗汉帮助才免饿鬼。
