# 佛陀独受岁之事

昔佛游句潭弥国时，天魔部党入大众诤讼诽谤，比丘离散。夏三月过后，佛惟与阿难共受岁。净居天子问佛何以独自受岁，佛告以昔出家、诣贝多树、今受岁皆以天子为证。
