# 菩萨身无毁定正受与觉悟的变换示例

菩萨于内身正受而于外身觉悟，于外身正受而于内身觉悟；以一身正受以多身觉悟，以人身正受以鬼神身觉悟，乃至于一尘中正受而现于无数刹土尘中觉悟等。
