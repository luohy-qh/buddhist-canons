# 毗梨耶波罗蜜菩萨种性相

菩萨性自精进，晨起夜寐，不乐眠卧，所作精勤不舍，思惟究竟，创始坚固，事未成不中废，于第一义心不退没，不自轻言，堪能勇猛，摧伏邪论，答难问，堪耐苦事。
