# 讲经缘起：新旧两本与讲者

片段记载讲经之始起竺法护，护公之后释安竺汰之流唯讲旧本。罗什至长安翻新法华后，道融讲之开为九辙，时人呼为九辙法师。讲新法花始乎融也。
