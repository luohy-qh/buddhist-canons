# 观世间如幻喻

观一切世间如缋像中而有高下，如梦所见端正女色，如石女人忽梦己身诞育于子，如乾闼婆城内诸所施为，如旋火成轮，如空中垂发，如幻化所作人马等形树林花果，如浮云之影，如奔电之光，皆是虚伪非真实有。
