# 田圆觉寄送法物

本国僧田圆觉自唐开成五年至大中九年，与圆珍共同抄写圣教绘曼茶罗，十年后送圆珍至天台，秋月出游往广州，附送天竺贝多树柱杖、广州班藤柱杖、琉璃瓶子等信物。
