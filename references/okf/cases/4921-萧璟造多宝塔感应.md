# 萧璟造多宝塔感应

唐国子祭酒萧璟依经文作多宝塔，其兄子铨得檀木浮图盖及鍮石佛像，盖与塔、像与塔皆相称；像内有舍利百余枚，曾试锤打不破，后舍利复归原处。
