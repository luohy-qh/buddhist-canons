# 如来化现龙王伊罗钵多罗

如来以龙王伊罗钵多罗为例，其住于须弥金福山边七宝宫殿，化身七万由延三十二头，每头七牙，牙上有宝浴池，池生莲华，华有玉女，作倡伎乐，雨七宝，以此比喻菩萨以四等心加被众生，雨七觉意无穷法财。
