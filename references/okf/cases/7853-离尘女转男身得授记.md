# 离尘女转男身得授记

会中有一女人名离尘，心生信解，自称已灭悭嫉，发菩提心，愿转身得成男子。发此言已即得转成男子之身，佛为授不退转阿耨多罗三藐三菩提记，当得成佛号除一切烦恼如来。
