# 太戊修政与祥桑枯死

亳有祥桑谷共生于朝，一暮大拱，太戊问伊骘，骘曰妖不胜德，君之政其有阙欤；太戊修先王之政，明养老之礼，早朝晏退，问疾吊丧，三日而祥枯死，三年远方重译而至者七十六国，商道复兴，号称中宗。
