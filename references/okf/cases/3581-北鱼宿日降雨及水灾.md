# 北鱼宿日降雨及水灾

北鱼宿日五月初雨堕十五斛，不宜下田，当修高田，天大淋雨，诸河漏溢，则有水灾，漂坏下田，高田独茂，时有三病：咽疾、脐痛、风痒。
