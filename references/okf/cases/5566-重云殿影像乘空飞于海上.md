# 重云殿影像乘空飞于海上

陈武帝崩，兄子茜嗣业，取梁武重云殿珠珩饰辒辌车。时云气拥结，雷雨大作，火烈云中，重云殿影及金银二像等一时腾举，烟火挟之远灭。后有人见殿影像乘空飞于海上。
