# 下品上生造恶与往生

作众恶业（不谤方等），无有惭愧，临终遇善知识为说大乘经题、教称佛名，灭罪，化佛来迎，生宝池中，七七日花开，闻十二部经，发无上道心，经十小劫得入初地。
