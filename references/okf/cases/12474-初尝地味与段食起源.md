# 初尝地味与段食起源

众中有一有情，禀性耽嗜，以指端尝地味，生爱著，遂初受段食。其余有情见之亦学食，身渐坚重，光明隐没，幽暗生起。因食量不调，形色损减，互相告语，生憍慢起不善根，地味遂灭。
