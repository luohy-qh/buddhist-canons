# 弟子入曼拏罗受法案例

弟子沐浴洁净后，以帛覆面，手持一花，念三宝名，由阿阇梨引入曼拏罗令发誓愿，抛花认本尊，阿阇梨说宝杖印及三宝秘密三昧印，以香花水灌顶，授与明王心大明，弟子受已作礼布施。
