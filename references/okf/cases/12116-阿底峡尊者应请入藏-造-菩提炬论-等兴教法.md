# 阿底峡尊者应请入藏，造《菩提炬论》等兴教法

因菩提光等殷勤迎请，阿底峡尊者至哦日铎，总集一切经咒要义束为修行次第，造《菩提炬论》等，并在哦日、聂塘、卫藏等处开示经咒教典教授。
