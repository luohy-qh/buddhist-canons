# 从大慈心到菩提心的因果链

大慈起于无谄诳心，无谄诳起于平等心，平等起于无彼无此无诸异见，异见起于善念恭敬，善念恭敬起于菩提心，菩提心起于六波罗蜜，六波罗蜜起于方便智慧，方便智慧起于不放逸，不放逸起于修三善业，三善业起于修十善业，十善业起于善摄身口意，三善业起于善思惟，善思惟起于善思观行，善思观行起于忆持不忘。
