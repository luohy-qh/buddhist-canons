# 下品中生人灭罪案例

一生具造佛法中罪（破斋破戒、食用佛法僧物、不生惭愧），得病欲死，地狱众火一时俱至，遇善知识为说弥陀佛身相功德国土庄严，闻已即除八十亿劫生死之罪，地狱即灭。
