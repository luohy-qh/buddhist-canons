# 昙翼祈像及长沙寺金像灵瑞

晋太元中，昙翼于荆州造长沙寺，洁斋祈像，二月八日像现城北，迎入寺中。像后被刀击，胸有刀迹。后罽宾僧认是育王像。梁武屡迎，像终南下，放光不绝。后寺烧，像独存。
