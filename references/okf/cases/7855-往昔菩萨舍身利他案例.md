# 往昔菩萨舍身利他案例

如昔为萨埵时，见饿虎欲食子，投崖舍命济彼饥；又昔为能舍王，一切所求皆充足，乃至身命及珍财；又昔为月光王，有恶眼婆罗门来乞头，即便舍头而施与。
