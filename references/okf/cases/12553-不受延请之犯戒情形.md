# 不受延请之犯戒情形

菩萨若因憍慢、嫌恨、恚恼而不受他人延请，是染违犯；若因懒堕懈怠忘念无记，是非染违犯；若因疾病、气力不足、狂乱、道有怖等因缘，则无违犯。
