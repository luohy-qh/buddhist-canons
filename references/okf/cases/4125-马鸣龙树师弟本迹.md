# 马鸣龙树师弟本迹

迦叶佛时，长者轮秀与女殊他各献七宝，所求十七日后满足。殊他梦日轮、月轮入腹，后生二子名日珠、月镜，即马鸣菩萨与龙树菩萨。二人常愿生生互为师弟子，建立正法。马鸣过去成佛号大光明佛等，龙树号妙云相佛等。
