# 佛会中是否有缘觉——两种说法

一说秘密即有、显现即无（智度论：初转法轮有三乘人各得其果是秘密众，大严经说五百辟支佛入灭是显现众）；一说独觉有本性独出与先声闻后独觉二种（旧婆沙）；一说独觉有麟角喻（独出无二）与部行（多人并出）二种（新婆沙、杂集论）。
