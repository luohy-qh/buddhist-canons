# 比丘特定日期诵咒除罪

比丘于月八日、十三日、十四日、十五日，洗浴护净，一日一夜不食或唯食三种白食，右绕佛塔诵此陀罗尼一百八遍，百千劫罪及五无间皆得除灭，除盖障现身满愿。
