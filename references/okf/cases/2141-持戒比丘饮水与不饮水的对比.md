# 持戒比丘饮水与不饮水的对比

两新学比丘途中遇旱，一人为见佛而饮有虫水，一人守戒不饮致死，死后生天，先至佛所；饮水者后至，佛示持戒生天者已先到，并说偈强调持戒胜于见佛。
