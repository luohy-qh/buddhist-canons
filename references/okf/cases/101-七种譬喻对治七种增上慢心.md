# 七种譬喻对治七种增上慢心

火宅喻对治颠倒求功德；穷子喻对治声闻一向决定；云雨喻对治大乘一向决定；化城喻对治实无谓有；系宝珠喻对治散乱；轮王解髻中明珠喻对治实有功德；医师喻对治实无功德。
