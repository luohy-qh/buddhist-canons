# 五通比丘犯欲堕贱

安息人至罽宾国为比丘，得五通后现神足，师诫之。比丘恚师，飞回本国受王供养。后与青衣有欲，腹大被逐。出外劫人，被擒，王赦之。后屠家槌牛，骨挑中眼，成盲丐。师率五百弟子来省，众得六通。
