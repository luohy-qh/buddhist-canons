# 五百释子比丘发狂案例

五百释子比丘在竹林修阿练若法，因听闻波斯匿王太子毗琉璃与五百长者子作象戏，莲华黑象声如霹雳，导致禅难提、优波难提等心惊毛竖，于风大观发狂痴想，从禅定起如醉象奔不可禁制。
