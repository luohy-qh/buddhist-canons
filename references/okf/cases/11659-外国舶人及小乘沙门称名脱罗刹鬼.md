# 外国舶人及小乘沙门称名脱罗刹鬼

外国百余人从师子国泛海向扶南，忽遇恶风堕鬼国，一舶众人怖称观音，小乘沙门不信，鬼索此沙门，沙门狼狈学称亦得免脱。
