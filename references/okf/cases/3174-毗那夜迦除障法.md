# 毗那夜迦除障法

蚁坟土作毗那夜迦形，大寒林中立坛，置形于佉驮罗木长十指，和毒药及血，进佉驮罗木火中满万遍，夜半形作声得其悉地，后作毗那夜迦法皆成就不被恼乱。
