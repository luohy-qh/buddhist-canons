# 降龙大师诚惠生平

降龙大师俗姓李氏，讳诚惠，蔚州灵丘县人，因父母祈五台山文殊而感孕出生，后出家受具，曾在龙宫池结庐修道，驯龙入瓶，后受王子寺僧请住寺，后唐庄宗赐号广法大师，终后敕赐谥号法雨大师。
