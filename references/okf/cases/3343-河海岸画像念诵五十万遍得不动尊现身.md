# 河海岸画像念诵五十万遍得不动尊现身

将画好的不动尊像于河海岸如法涂坛安像，行者著赤色衣，寂静安心乞食为活，像前念诵五十万遍，夜中取詹末一万段一诵一掷火中烧，满已不动尊自现，行者证三摩地共诸菩萨同位。
