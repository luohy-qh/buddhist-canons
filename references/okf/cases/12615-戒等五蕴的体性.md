# 戒等五蕴的体性

婆沙三十三出戒等五蕴体云。云何无学戒蕴答无学身．语律仪。云何无学定蕴。答无学三三摩地。云何无学慧蕴答无学正见智。云何无学解脱蕴。答无学作意相应心。已胜解．今胜解．当胜解。云何无学智见蕴．答尽智．无生智。
