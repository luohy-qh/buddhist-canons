# 凡夫佛性如杂血乳

依迦叶品，凡夫佛性如杂血乳；须陀洹、斯陀含断小烦恼，所有佛性如出血乳；那含如酪；罗汉如生苏；缘觉菩萨如熟苏；佛如醍醐。
