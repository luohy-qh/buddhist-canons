# 沙弥神足戏水度王

舍卫国王不信佛，见沙弥脱袈裟作屋戏而讥之。夫人未语竟，沙弥取水后以神足挑三百瓶虚空，飞逐入精舍。王见大欢喜，诣佛悔过，发无上道意。
