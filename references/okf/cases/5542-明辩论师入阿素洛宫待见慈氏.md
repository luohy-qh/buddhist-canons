# 明辩论师入阿素洛宫待见慈氏

明辩论师（婆毗吠伽）在观自在前绝粒饮水三年，后依观自在指引，在南大山岩执金刚神处诵咒三年，得神授方后以咒芥子击石壁，石壁洞开，论师与六人进入，石门复闭如壁，以待慈氏出现。
