# 耶只从外道皈依佛受五戒

婆罗门耶只本信九十六种外道，后闻佛道大慈普济，舍外道归命于佛，受五戒（不杀生、不盗、不淫、不两舌恶口妄言绮语、不饮酒）及三自归，持斋七日而去。
