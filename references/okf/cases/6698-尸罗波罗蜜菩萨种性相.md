# 尸罗波罗蜜菩萨种性相

菩萨身口意业柔软，不增恶行，不乐杀生，悔过，不恐怖众生，仁贤慈爱，恭敬尊长，善知机宜，含笑先意，知恩报恩，正直，受如法财，喜乐修福，怜愍苦众，重善，于轻罪恐怖，与如法事同事，遮十不善，实语，不粗言等。
