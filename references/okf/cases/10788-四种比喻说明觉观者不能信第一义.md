# 四种比喻说明觉观者不能信第一义

以啖苦味者不知甘味、贪欲者不知无欲乐、分别者不知无分别乐、见闻觉知者不知离见闻觉知乐、取我相者不知无我乐，比喻觉观之人不能知第一义相。
