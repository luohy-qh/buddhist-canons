# 造像法

像形端立，象头人身，左牙出右牙折，面少向左，鼻向外瘘，有六臂：左上手把刀，次手把果盘，下手把轮；右上手把棒，次手把索，下手把牙。造像不得违价，足数付之。可铸、刻、画。
