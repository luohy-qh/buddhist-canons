# 文殊师利化现外道度化萨遮尼乾子徒众

文殊师利化作五百异道师徒，前往萨遮尼乾子处，以谦卑姿态取得信任，融入其徒众，并逐渐赞叹三宝功德，最终使外道归信佛陀。
