# 外国众人遇恶风堕鬼国称观音得脱

外国百余人从师子国泛海向扶南，遇恶风堕鬼国，众人怖称观音，一小乘沙门不信观音不肯称名，鬼索此沙门，沙门狼狈学称亦得免脱。
