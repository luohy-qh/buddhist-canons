# 戒日王日常行仪

王巡省方俗不常其居，行必四兵导引，泛舟乘象击鼓吹螺，象军八万以威四远，惟两三月不外游行；宫中每日饭沙门千人与婆罗门五百，日分三时一时理务，两时营福，绝血食，日例一顿。
