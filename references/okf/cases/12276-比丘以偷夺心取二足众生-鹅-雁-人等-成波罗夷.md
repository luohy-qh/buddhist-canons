# 比丘以偷夺心取二足众生（鹅、雁、人等）成波罗夷

比丘以偷夺心取笼中鹅、雁、孔雀、鹦鹉及人等二足众生，选择后取直五钱即波罗夷；穿笼取鸟直五钱亦波罗夷；偷人担去过二踔或共期行过二踔亦波罗夷。
