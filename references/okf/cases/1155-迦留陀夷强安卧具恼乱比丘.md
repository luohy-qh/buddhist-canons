# 迦留陀夷强安卧具恼乱比丘

迦留陀夷为人恶眠。墙[口*動]大唤。先与一比丘宿。令他失眠。饮食不消。身体发创。复于异时二人一处。迦留陀夷知他先住。不问彼人强安卧具。
