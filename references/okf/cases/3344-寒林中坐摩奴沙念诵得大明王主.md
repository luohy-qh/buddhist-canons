# 寒林中坐摩奴沙念诵得大明王主

黑月八日夜，于寒林中取母耶摩奴沙，坐其上念诵满一万遍，摩奴沙动耳开口出大开敷莲华，把取后能令己身如十六童子，发如莲环，升空游于梵天，得大明王主。
