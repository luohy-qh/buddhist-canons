# 外道尼乾子因闻法得证忍案例

过去善胜调伏佛说法时，诸外道为恼而来，闻法称善但不敬；以此因缘六万劫不生三恶道，唯生人天，不值善知识。今于如来所生欢喜恭敬，得受记及无生法忍。
