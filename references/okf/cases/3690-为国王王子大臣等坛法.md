# 为国王王子大臣等坛法

于城外简择好地处，量取三十二肘穿深六尺，去恶物，取高黄土筑令平实，用野牧牛粪和香水泥涂，上安白盖，四面悬缯幡幢，开四门，用七色宝末界坛，画千叶莲花及诸佛菩萨像。
