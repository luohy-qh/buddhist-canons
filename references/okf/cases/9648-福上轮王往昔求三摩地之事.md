# 福上轮王往昔求三摩地之事

世尊说，过去有转轮王名福上，统领阎浮提，闻髻幢佛说三摩地，心生欢喜，舍国出家，经八万岁常习三摩地，昼夜不懈。福上轮王即今安乐国无量寿佛。
