# 智琮烧香感得舍利瑞光

显庆四年智琮入塔行道，至十日三更臂上安炭烧香，专注无想，忽闻振裂声，见瑞光流溢，塔内三像足各放光，后获舍利一枚，又得七枚，盘中一枚独转放光。
