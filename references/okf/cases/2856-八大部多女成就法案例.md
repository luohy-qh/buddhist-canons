# 八大部多女成就法案例

有八大部多女，一名部底二名尾沙尼...此八部多女能为持诵之者作成就故。若有行人于夜分中往其树下，持诵真言至第三日满八千遍，然后作大供养烧安悉香，复诵真言至中夜时，尾部沙尼部多女即现人前，用白檀香水献阏伽，令彼欢喜。若欲为母，彼即为母如子养育，日给衣服饮食可赡百人；若欲为妻，日给金钱一千；若为姊妹，百由旬内所有端正女人即为将来。
