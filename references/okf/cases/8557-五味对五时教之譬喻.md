# 五味对五时教之譬喻

以牛出乳喻佛出十二部经（华严时），乳出酪喻从十二部经出九部修多罗（鹿苑时），酪出生酥喻从九部出方等（方等时），生酥成熟酥喻从方等出摩诃般若（般若时），熟酥成醍醐喻从摩诃般若出法华（法华涅槃时）。
