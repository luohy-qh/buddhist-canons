# 刘萨何预言石像出山及身首离合

元魏时，刘萨何预言番禾郡御谷当有像现，像身端严则世乐，有阙则世乱。后像身出，无首，安首复落。至周，于城东得佛首，安之符合。建德初像首频落，后灭法，预言皆验。
