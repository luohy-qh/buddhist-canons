# 大佛塔与钟台

方池正北有七层大佛塔，纯以异石建造，状丽宏异，塔上有金铃，微风常动清响和彻。塔下有迦叶佛爪发舍利。塔傍左右立二钟台，左边为他化天王第三子无畏所造，钟及台并颇梨所成；右边为兜率天王所造，钟及台并金银所成。二钟各受五十斛，不常鸣，每至十方诸佛集始鸣，声闻百亿世界。
