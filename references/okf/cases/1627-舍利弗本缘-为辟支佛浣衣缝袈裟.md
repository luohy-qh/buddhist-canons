# 舍利弗本缘：为辟支佛浣衣缝袈裟

舍利弗前世为仙闲居时，见辟支佛沙门，为之浣衣、缝袈裟、数数作礼，并发愿得智慧，以此功德五百世作沙门，最后值佛成阿罗汉。
