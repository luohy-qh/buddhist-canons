# 令竖子相憎与相怜法

相憎法：多罗叶画男女形书名相背，取线缠，以鼠狼毛等烧熏，诵陀罗尼一百八遍，安劫波罗中埋尸林。相怜法：于多罗叶上画夫形书名相合，取白线系，以雀儿毛蛇皮烧熏，诵陀罗尼一百八遍。
