# 达尼迦盗王材案

比丘达尼迦以国王登位时曾施一切草木于沙门为由，取王材。国王未直接惩罚，令其回佛法王处受治。此事导致恶名流布，佛陀因此集僧结戒，规定盗五钱以上得波罗夷。
