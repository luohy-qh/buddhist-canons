# 才发意转法轮菩萨印之应用

以印置于身前坛上，即成莲华部调伏大曼荼罗；安于心上，自身成大曼荼罗；触本尊像，彼像成大曼荼罗；置身前空中，满虚空界成大曼荼罗。能除越法、误失三业、破三昧耶戒等诸过。
