# 么啰谤毁三宝者法

以人髑髅末作彼人形，面北于赊摩赊那，左脚踏小指刺，诵七日日三时，可令其大疟疮疱至死；若于佛法生净信则息忿怒，以香水灌沐佛像念诵起慈心可止息。
