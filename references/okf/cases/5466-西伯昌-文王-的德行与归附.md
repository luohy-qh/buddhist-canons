# 西伯昌（文王）的德行与归附

西伯昌行政善，诸侯来归之；伯夷叔齐闻其善养老归焉；太颠、闲夭、散宜生、鬻子、辛甲之徒皆归之；吕望年八十余钓渭水，西伯载之归尊为太公；虞苪争田质于西伯，入境见耕者让畔行者让路，惭而还，汉南诸侯闻而归者四十余国。
