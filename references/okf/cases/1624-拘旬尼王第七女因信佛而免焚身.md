# 拘旬尼王第七女因信佛而免焚身

王第七女原事梵志后信佛，梵志以王十梦欲陷女，女愿焚祠天，临期请佛说法，六日间使城人及王宫皆见谛，王知梵志欺诈，梵志不得已出家得阿罗汉。
