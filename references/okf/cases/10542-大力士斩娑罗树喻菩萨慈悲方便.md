# 大力士斩娑罗树喻菩萨慈悲方便

文殊师利以大力士持利刀斩娑罗树，树虽断犹住不堕，喻菩萨有大慈悲大方便智，虽知我见而不证果；天降大雨树还生，喻菩萨大慈悲润洽还生三界利益众生；大风摧树堕地，喻大智慧猛风吹堕道场毕竟永灭。
