# 光目女救母案例

光目女之母因杀害毁骂二业堕入恶道受极大苦。光目女依罗汉指点，志诚念清净莲华目如来并塑画形像，后发广大誓愿救拔地狱及三恶道众生，其母因此脱难并最终成佛。
