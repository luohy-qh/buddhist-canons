# 檀波罗蜜菩萨种性相

菩萨性自乐施，等施不倦，欢喜无悔，常叹施功德，劝令行施，随喜，恭敬奉施，为说如法事，施无畏，受寄无差，不抵债，平等分财，教离贪，舍好财，乐修胜业，远离酒色戏事等。
