# 育王第四女造瑞像因缘

育王第四女貌丑，图佛形后发愿，感佛现异形。所造像分布北山玉华、荆州长沙、杨都高悝、崇敬寺等处，诸像面莫匪女形。崇敬寺地本是战场，故诸鬼神携以镇之。
