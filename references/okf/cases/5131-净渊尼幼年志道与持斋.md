# 净渊尼幼年志道与持斋

净渊本姓时，钜鹿人。幼有成人之智，五六岁时尝聚沙为塔刻木作像。二十出家，恋慕膝下，不食不寝，饮水持斋，谏晓不从，终竟七日。自尔之后蔬食长斋，戒忍精苦不由课励。师友嗟敬远近称誉。
