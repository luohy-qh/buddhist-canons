# 龙女善女献如意宝珠

龙女善女因先世罪障深重，贪恚愚痴而不殖智种，受生为湖水中女身，虽无热恼及诸苦患犹居畜类鳞族之中。她以偈赞佛后，持一从龙渊底九重苍海中水精宝箧中得来的如意宝珠奉献如来，愿世尊受其宝珠，以利益有情。
