# 明王解脱魔民及食血等众生

若魔、魔民以恶心相向，或自在天、夜叉、夜叉女等，乃至一切食血者、食肉者、食脂膏者、食精气者、食涎沫者，所执所缚悉皆解脱；男女受胎、处胎、初生皆得安隐。
