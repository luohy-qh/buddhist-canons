# 吉槃陀女沽酒堕地狱案例

舍婆提国吉槃陀女为人沽酒，钗梳落在井中，八万罗汉饮水昏醉不识如来；该女堕锯床地狱，锯身百段，后身长三尺、颜色青黑、无目无鼻等，皆由沽酒。
