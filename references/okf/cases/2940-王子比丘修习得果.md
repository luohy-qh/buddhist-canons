# 王子比丘修习得果

往古星持轮王之子不思议功德最胜，于宝胜威宿劫王佛所初得此陀罗尼，经七万岁舍身命财王位，独处闲静，于九十九亿百千那由他诸如来所闻经受持，后出家教化众生，即今无量寿佛。
