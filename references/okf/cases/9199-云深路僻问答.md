# 云深路僻问答

师问‘云深路僻高步何来’，僧答‘天无四壁’，师问‘踏彼多少草鞋’，僧喝，师问‘一喝两喝又作么生’，僧说‘尔看者老和尚’，师说‘拄杖不在，且坐吃茶’。
