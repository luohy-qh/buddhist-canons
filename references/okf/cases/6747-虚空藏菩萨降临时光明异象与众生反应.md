# 虚空藏菩萨降临时光明异象与众生反应

西方如意宝珠现，光明暎彻，人天声闻菩萨光明皆灭，唯除如来及如意宝珠；众生各不相见、触、闻、觉，唯见如来色相光明如意宝珠。住十地等菩萨不怖畏，余众惊怖。
