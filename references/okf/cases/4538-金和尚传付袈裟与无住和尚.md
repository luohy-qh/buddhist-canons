# 金和尚传付袈裟与无住和尚

金和尚遣董璇将袈裟一领及衣物密送与无住禅师，并传语“善自保爱努力努力。未出山时。更待三五年间。自有贵人迎汝即出。”金和尚言此衣是则天皇后与诜和上，诜和上与唐和上，唐和上与吾，吾传与无住禅师。
