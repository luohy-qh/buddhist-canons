# 童子因虫食阴习口中行淫被判波罗夷

阿阇世王之子因虫食阴，养者以口含阴治病，后习以为常，转强牵他人行口中淫，优波离问佛：比丘与比丘、沙弥、白衣、比丘尼、外道等共口中行淫，是否犯波罗夷？佛答：比丘俱波罗夷，沙弥驱出，白衣、外道则不论。
