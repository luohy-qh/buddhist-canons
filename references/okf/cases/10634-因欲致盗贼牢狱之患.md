# 因欲致盗贼牢狱之患

因欲之患，破他门户，断人寄饷，凿人垣墙，夜行作贼，或逃亡斗争放火，国主觉得，闭著牢狱，截耳鼻手足，或断头首，或镬汤煮、沸油洒体。
