# 罗睺罗为耶输陀罗之子

法华经称罗睺罗为耶输陀罗之子，与其他经称瞿夷之子不同，因耶输是第二夫人，瞿夷是第一夫人，罗睺罗实为第二夫人儿，瞿夷是其大母。
