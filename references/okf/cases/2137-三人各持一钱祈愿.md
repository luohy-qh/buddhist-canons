# 三人各持一钱祈愿

三个卖樵穷人，四月八日灌像时各持一钱布施祈愿：一人愿饶财宝，一人愿知医方，一人愿长寿。佛说三人各得所愿，后皆成为佛弟子得阿罗汉道。
