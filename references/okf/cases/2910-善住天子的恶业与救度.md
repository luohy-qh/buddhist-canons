# 善住天子的恶业与救度

善住天子因放逸嬉戏，被预告七日后命终，于阎浮提七返恶道（猪、狗、狐、猴、蛇、乌、鹫），后入地狱，出得人身又盲瞽贫穷。释天求救，佛说此陀罗尼，善住闻后恶业除灭，寿命增长。
