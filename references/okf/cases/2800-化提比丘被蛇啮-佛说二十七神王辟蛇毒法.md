# 化提比丘被蛇啮，佛说二十七神王辟蛇毒法

化提比丘治护屋室时被黑蛇啮，闷绝躄地。佛令阿难以神水濆洒，并说二十七神王辟蛇毒法，化提即愈。阿难告诸比丘，安居住止处应说此言，蛇毒七岁不复啮人。
