# 国子祭酒孛木鲁翀与帝师抗礼

西僧帝师至，朝臣一品以下咸郊迎俯伏进觞，师不为动。惟国子祭酒孛木鲁翀举觞立进曰：帝师释迦之徒，天下僧人师也；予孔子之徒，天下儒人师也。请各不为礼。帝师笑而起，举觞卒饮。
