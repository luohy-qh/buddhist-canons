# 须摩提发愿即现神通感应

须摩提发愿若审实能行四十事，则三千大千国土六反震动、雨天华、乐器自鸣，应时即现。后又发愿变为男子，即成沙弥；发愿未来佛国无魔事、泥犁、女人态，身即如年三十；发愿会众作金色，即皆金色。
