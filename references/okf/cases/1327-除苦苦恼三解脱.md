# 除苦苦恼三解脱

除苦苦恼即于嗔解脱，除坏苦恼即于贪解脱，除行苦恼即于痴解脱。于嗔解脱是解脱欲界，于贪解脱是解脱色界，于痴解脱是解脱无色界。
