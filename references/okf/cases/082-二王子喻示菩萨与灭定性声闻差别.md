# 二王子喻示菩萨与灭定性声闻差别

譬如二王子形貌无异，同受王乐，其中一善达王法技艺，另一不如是，技艺胜者非受用乐处。菩萨住无漏戒中与灭定性声闻亦如是，彼复有胜事。
