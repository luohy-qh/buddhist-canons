# 结金刚甲胄印披大誓庄严金刚甲胄

二手虚心合掌，头指屈拄中指背上节，大指押中指中节文，触额、右肩、左肩、心、喉各加持一遍，即是披大誓庄严金刚甲胄，天魔及作障者不敢凌逼。
