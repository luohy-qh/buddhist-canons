# 梦相证明灭罪

凡夫取梦相，若见一梦即灭一逆，见五梦即灭五逆。例如梦见自身欲渡大河上大桥行，定得度脱；梦见自身人与洗浴天雨其身，定得清净；梦见入沙门大会中入次而坐，是真佛弟子；梦见入塔寺中见好大像及见菩萨，得正门；梦见自身得果食而食，还得果报。
