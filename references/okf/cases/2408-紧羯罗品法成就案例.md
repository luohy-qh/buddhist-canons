# 紧羯罗品法成就案例

持咒者日日得一百金钱，用于佛法僧，不应悭吝，并需恒常供养、先出食分，常实语法语、舍贪嗔痴、生慈心，敬事观自在菩萨，则五百眷属恒时供养不乏。
