# 诸国以武力威胁夺取舍利

诸国回应城人：“若分者善。若不与者我等当以强力夺取。”城人答：“徒事斗诤终不可得。”阇王复派雨行大臣驰兵请分，若不分则“加兵力强夺将去”。
