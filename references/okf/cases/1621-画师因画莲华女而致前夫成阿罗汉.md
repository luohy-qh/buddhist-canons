# 画师因画莲华女而致前夫成阿罗汉

阿育王因画师所画女子娶为皇后，皇后闻前夫香而悲，王怒录其前夫，前夫已为阿罗汉，身香胜莲华，因前世叉手赞叹菩萨并烧香供养得果。
