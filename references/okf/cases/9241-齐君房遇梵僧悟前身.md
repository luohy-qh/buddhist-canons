# 齐君房遇梵僧悟前身

齐君房在孤山遇梵僧，食枣后悟前身为洛中同德寺讲法华经法师；梵僧指出因广说异端、戒珠有缺、禅不兼修而致冻馁，并授镜照知诸事；齐君房后祝发为僧改名镜空。
