# 闲居菩萨因恐惧而来

菩萨因恐惧聚会习淫怒痴、贡高、贪嫉、色声香味细滑、四魔、常见、乐想、实想、我想、吾我、狐疑、恶知识、爱欲、心意识不调、诸盖、自见身、财利、求处见、念无念、妄见妄闻、沙门垢、种种行、三界、地狱畜生饿鬼、八难、五道、卒念诸不善法等而来闲居，以解脱恐惧得佛道无畏。
