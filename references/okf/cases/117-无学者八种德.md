# 无学者八种德

无学者有八种德：一所作毕竟（应作已作），二毕竟过于应作已作（所作已办），三远离三昧障（弃舍重担），四舍离所受重担（逮得己利），五证涅槃（尽诸有结），六过三界（正智心得解脱），七依不颠倒受教修行（一切心得自在已到彼岸），八如实修行四如意足（皆是阿罗汉）。
