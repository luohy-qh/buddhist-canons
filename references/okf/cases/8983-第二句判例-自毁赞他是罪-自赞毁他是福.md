# 第二句判例：自毁赞他是罪、自赞毁他是福

如知时世风俗憎自赞毁他、敬自谦扬他，或知毁彼彼必呰我，巧求自高而自毁赞他，是为重罪；若知他人所执非理、自内所解是道，直欲建立佛法饶益有情而自赞毁他，是为大福。
