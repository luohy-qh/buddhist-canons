# 远离邪见得十种功德

远离邪见获得自心安善、深信因果、不为身命作罪业、得贤圣位、不迷善法、不堕恶趣、行圣道得最上福、离邪法、离身见、见罪性空、天上人间正行不阙等十种功德。
