# 太子无念德首奉行此持得见九十亿佛

往昔有佛宝首曜王，临灭度时转轮王太子无念德首从佛得闻此持而奉行，七千年未睡卧、未起身爱、未念财利，一心念行，于是见九十亿佛，悉闻说法皆从受持。
