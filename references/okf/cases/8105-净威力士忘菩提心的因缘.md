# 净威力士忘菩提心的因缘

过去世迦迦孙佛时，善财婆罗门（即今净威力士）因憍慢增上，不往佛所，与诸婆罗门诤竞，以此恶业忘失菩提心，但以菩提心本善根故不堕恶道。
