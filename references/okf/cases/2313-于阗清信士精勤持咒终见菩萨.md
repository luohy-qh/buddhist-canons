# 于阗清信士精勤持咒终见菩萨

昔于阗国有一位清信士，精心持念观自在菩萨真言多年，虽未立即见圣，但执志不易，倍复精勤。后闻空中声指引，往邻国谒王，发现王即观自在菩萨化身。菩萨为度化刚强众生，分身为王，以刑戮教化，所刑戮者皆是化人。行者被王留住数月，通悟大乘后辞归。
