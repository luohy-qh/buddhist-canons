# 菩萨往生安乐无退转难

菩萨于七地中得大寂灭，上不见诸佛可求，下不见众生可度，欲舍佛道证于实际。尔时若不得十方诸佛神力加劝，即便灭度与二乘无异。菩萨若往生安乐见阿弥陀佛，即无此难。
