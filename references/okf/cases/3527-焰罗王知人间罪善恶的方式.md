# 焰罗王知人间罪善恶的方式

王常见檀拏幢头少忿怒之面，知人间罪轻重善恶：重罪者从其口出火光，光中黑绳涌出，见木札知其姓名料记；作善者白莲花从口开敷，香普薰大山府君五道将军王。
