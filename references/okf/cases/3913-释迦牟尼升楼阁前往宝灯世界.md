# 释迦牟尼升楼阁前往宝灯世界

释迦如来从座而起，诣升七宝楼阁以手触之，楼阁中出金刚师子妙座，座生妙莲华，如来于莲华上敷座而坐，接诣虚空往宝灯世界。
