# 阎浮提及诸洲儿饮乳量及寿命

弥勒答佛问：阎浮提儿三岁饮乳一百八十斛，东弗于逮一千八百斛，西拘耶尼八百八十斛，北郁单曰无乳；阎浮提人寿命百岁，东弗于逮五百岁，西拘耶尼二百五十岁，北郁单曰千岁，中阴众生寿命七日。
