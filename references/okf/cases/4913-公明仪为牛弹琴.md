# 公明仪为牛弹琴

公明仪为牛弹清角之操，牛伏食如故，转为蚊虻孤犊之声，牛则掉尾奋耳而听，以此比喻对盲者说五色、为聋者奏五音，说明对不理解佛经的人需用其能懂的诗书引导。
