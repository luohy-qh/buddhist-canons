# 鞞罗羡那王子出家七日命终生天

毗舍离城王子鞞罗羡那（勇军）贪五欲乐，佛记其七日命终，若不求出家将堕地狱。王子闻后于第七日舍家出家，一日一夜持净戒，命终后依次生于四天王天、三十三天、焰天、兜率天、自在天、他化自在天，于六欲天中往返七次，最后生人中富乐家，厌世出家成辟支佛。
