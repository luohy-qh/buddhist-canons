# 龙众执魅的具体表现

西方龙主广目天王描述龙众执魅时，众生会现常餲哕、喘息长嘘、身常冰冷、口吐涎沫、多睡眠、身体如蛇光滑坚硬、心勇猛不惧生死、能驰走无困乏、爪甲长利、手爮其地令如窟穴等相。
