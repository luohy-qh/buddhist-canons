# 弥勒天寿与人间年数换算争议

人间四百岁为彼天一日一夜，彼天寿四千岁，当人间五十六亿七千万岁。但瑜伽论等算出五千七百六十亿岁，基师认为亿有二种（十万为亿、百万为亿），经中亿字上应加七字，译家有误。
