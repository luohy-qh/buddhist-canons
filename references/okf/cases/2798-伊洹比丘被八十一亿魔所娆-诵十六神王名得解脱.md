# 伊洹比丘被八十一亿魔所娆，诵十六神王名得解脱

昔伊洹比丘为八十一亿魔所娆，诵此十六神王名字，诸魔眷属颠倒堕落、匍匐离散。佛告比丘若有危厄恐怖之日，呼此神名即获吉祥。
