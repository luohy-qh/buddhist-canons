# 治病除障法

若有病患者，结龙王身印加持，观想婀字变成金色龙，忿怒光曜，令观病人即得除差。若有鬼魅疫病等，作一肘水坛，四角置瓶插生华，四方置四器盛香水，劝请龙王安住坛上，置剑启白，诵真言百八遍，执刀加被病人即得消灭。
