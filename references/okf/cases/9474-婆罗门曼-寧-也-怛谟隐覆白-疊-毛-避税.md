# 婆罗门曼[寧*也]怛谟隐覆白[疊*毛]避税

婆罗门将新、故二白[疊*毛]置伞柄中，经税场不纳，柱铃自动；守税人搜出后不纳王，挂柱奉天神，婆罗门复取前行。
