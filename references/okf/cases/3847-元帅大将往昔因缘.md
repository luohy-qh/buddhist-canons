# 元帅大将往昔因缘

彼大将于空王如来所修菩萨行，末法时众生饥渴，其舍妻子眷属，著故破衣担水与食救人六十年；后遭狂贼执缚，临终发愿舍此身体作大力勇猛之神，摧伏恶贼恶人，救众生枉横。
