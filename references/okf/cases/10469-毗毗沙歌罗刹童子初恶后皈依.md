# 毗毗沙歌罗刹童子初恶后皈依

往昔有大悲所生智相幢如来住摩罗耶山顶，罗刹童子毗毗沙歌因怕失去捕食对象而欲驱佛，带众持械飞行虚空威胁，后因佛现神通被缚，经善友劝归依三宝发菩提心得脱。
