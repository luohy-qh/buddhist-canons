# 王子以肉济父母缘：割肉供亲

王子携父母妻子逃亡，粮食耗尽。王子欲杀妇存命，其子请求代母而死，并自愿日日割肉供父母食用，直至仅剩三脔。后因不悔之心，身体平复如故，并得帝释天护助，复国为王。
