# 羊车行菩萨喻

如有人为大事因缘欲过五佛刹微尘数世界，乘羊车而行，久受劳苦行百逾缮那，忽遇大风吹退八十逾缮那，终不能超越一世界。喻发心求无上正等正觉者，与声闻同住、亲近、习学声闻乘教，智慧微劣，退失无上智道。
