# 千二百五十人常随众由来

佛初成道度憍陈如等五人，次度优楼频螺迦叶五百人，伽耶迦叶、那提迦叶各二百五十人，舍利弗、目乾连各一百人，那舍长者子等二百五十人，总合一千二百五十人。皆名德高远常随佛。
