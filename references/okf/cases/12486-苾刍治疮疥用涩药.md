# 苾刍治疮疥用涩药

苾刍患疮疥，医人嘱服涩药，佛开许五种涩药（庵没罗、纴婆、赡部、夜合、俱奢摩），并规定捣碎水煮涂身、微日曝药、待干手摩入皮肤后沐浴、疮愈后残药应收举送病坊。
