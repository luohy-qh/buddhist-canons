# 盗稻纠纷与地主立王

一有情私盗他谷，经三次劝告不止，众人捉往众中毁辱。因此大众共集，简一颜色端正、智慧通达者立为地主，治罚有过者，养育无过者。众人所种之田，六分之中与其一分。由此因缘立为地主，得太同意、刹帝利名，号为大同意王。
