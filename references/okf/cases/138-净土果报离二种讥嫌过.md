# 净土果报离二种讥嫌过

净土果报离二种讥嫌过：一者体，二者名。体有三种：二乘人、女人、诸根不具人，无此三过故离体讥嫌。名亦三种，非但无三体，乃至不闻二乘女人诸根不具三种名，故名离名讥嫌。
