# 三性中根尘识差别：分别性、依他性、真实性

分别性根尘识从人法二妄心生，情有理无（如暗藤似蛇）；依他性从本识种子因缘生，因缘有法（如昼见实藤）；真实性根尘识是佛菩萨报身功德，二智种子生，法界所流证实无倒（如真金）。
