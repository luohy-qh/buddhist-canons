# 造百千俱胝层阁不如造小塔

能造百千俱胝层阁高广如帝释宝台奉施僧众，不如于如来涅槃后起小塔安置芥子许舍利、造小佛像，前者福聚不及后者百分千分乃至邬波尼杀昙分。
