# 懈慢国（疑城）

过此西方十二亿那由他刹有懈慢国，人欲往生阿弥陀佛国若从此过，多染著即愿生其中，遂不得到弥陀佛土。若见此不贪不爱，即得越过至安乐国。此即无量寿经所谓疑城。
