# 禅波罗蜜菩萨种性相

菩萨于法于义善思量无乱想，乐远离处，薄烦恼轻阴盖，不为恶觉恼乱，起慈心悲心，饶益众生，安忍难事，速受持诸法深义，念力成就，久修忆持。
