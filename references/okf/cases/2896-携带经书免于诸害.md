# 携带经书免于诸害

若人将经带身上或系于项，则毒、器械、水火、蛊毒、咒诅、夭横皆不能害，鬼魅见者欢喜，金刚部众守护，得宿命智，不生恶趣，不生贫穷下贱，得无量福德。
