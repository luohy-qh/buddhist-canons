# 病危死相案例

病痛相不可治时，口不知味，耳中不闻声，一切卷缩，脉投血肉，颊车张，上头掉影无有明，臀肉竖，眼黑色黑，大小便不通，节根解，口中上臛青，双哕计。设鶣鹊及一切良医并祠祀尽会，亦不能愈。
