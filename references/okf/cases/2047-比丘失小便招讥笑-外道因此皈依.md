# 比丘失小便招讥笑，外道因此皈依

有比丘在路上站立失小便，被行人讥笑。外道尼揵种见后，认为自己的裸行教法无人嘲笑，反而佛弟子因行为有威仪而被人议论，说明佛弟子法清净有礼仪，于是便皈依佛作沙门，得须陀洹。
