# 白芥子掷散各种场景的效力

白芥子掷于虚空止霜雹暴风；掷散四方禁缚虫兽；掷于火中火不能烧；掷于江河水不流；掷于商估中不被贼劫；掷于王宫门令国王大臣欢喜；掷于大众得供养；掷于他敌军众被禁止；掷于关戍身隐而过；散于苗稼不被虫伤；天旱掷龙池得大雨；暴雨时掷空雨止；掷冤家舍中不复有恨；掷城门内一切逼迫消灭。
