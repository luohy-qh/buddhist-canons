# 求那陀比丘患眼风痛湿虫，佛说六神章句

昔有求那陀比丘患眼风痛，又患湿虫痒痛难忍，佛为其说此神咒六王为护。若有末世诸比丘辈及清信士女，为虫所唼眼者，当为说此六神章句，无不得愈。
