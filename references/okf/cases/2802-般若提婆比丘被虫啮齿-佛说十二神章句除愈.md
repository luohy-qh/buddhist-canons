# 般若提婆比丘被虫啮齿，佛说十二神章句除愈

有比丘名般若提婆，诵习经法时为虫所啮齿。佛说是章句，以净水含水一口，牙临其上七遍诵此十二神王，吐水，如是四十九遍，虫便破散随水流迸，无不除愈。
