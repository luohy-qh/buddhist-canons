# 释迦与三千佛缘

释迦牟尼于妙光佛末法时闻五十三佛名，合掌欢喜，教他人辗转相教至三千人。此三千人因称佛名敬礼，得超越生死之罪，后分别于庄严劫、贤劫、星宿劫成佛。释迦在贤劫中为千佛第四。
