# 迦陵频伽鸟在卵中已异于众禽

迦陵频伽鸟住卵中时已与一切飞禽不同，因当发美妙音声；初发心菩萨虽住烦恼无明藏中，早与一切声闻辟支佛不同，因有回向善根说法方便。
