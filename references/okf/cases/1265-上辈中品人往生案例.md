# 上辈中品人往生案例

上品中生者，不必受持读诵方等经典，但善解义趣，第一义心不惊动，深信因果不谤大乘，临终阿弥陀佛与观世音大势志持紫金台来迎，坐紫金台往生，经一宿莲华开，七日得不退转，一小劫得无生忍。
