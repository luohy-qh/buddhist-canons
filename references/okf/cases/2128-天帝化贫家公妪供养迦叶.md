# 天帝化贫家公妪供养迦叶

天王释数下供养三尊，唯摩诃迦叶不肯受，本愿但欲度贫穷人；天帝化为贫家公妪，以粗食供之，实则盛满甘露，迦叶三昧观知，天帝言三尊福报无量，智者未尝厌足。
