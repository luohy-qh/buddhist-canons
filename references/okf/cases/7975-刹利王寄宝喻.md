# 刹利王寄宝喻

灌顶刹利大王病重寄宝藏于大臣长者，嘱其子还时付之；子还绍位后，大臣长者不还宝物。佛以此喻如来付嘱法宝，阿罗汉若不受嘱付还则有过。
