# 差摩竭得法及授记

差摩竭得不起法忍；五百比丘及五百清信士二十五清信女皆得立不退转地，寿终悉当生于西方无量寿佛清净国，常护持无数佛法，教化成就一切人民，使不退转，当于此土以次作佛。
