# 摩奈斯大龙徐雨喻如来开化纯淑类

摩奈斯大龙王先贮集云遍诸天宫，观察众人农业普备后徐降微渧，不欲烦恼众生；如来为大法王，为纯淑类雨甘露味，畅深奥典不令众生怀恐惧心。
