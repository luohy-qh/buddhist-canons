# 善住天子因恶口骂母及僧得恶报

善住天子过去因饥饿嗔恨母亲，骂母不如畜生；又作维那时吝惜僧食，骂客僧食屎尿、眼盲，故感得七返畜生身、常食不净、地狱受苦、无目等报。
