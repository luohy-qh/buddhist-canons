# 善明设供遇佛神变

善明欲设饮食供佛及大众，因财不足而忧。佛知其所念，神力使力办众有明士及诸天神王相助，屋室自然广长高大，千亿万座、百千亿万人饭具弹指间足办，食毕器具满则如故。
