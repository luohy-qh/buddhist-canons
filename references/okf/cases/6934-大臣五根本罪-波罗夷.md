# 大臣五根本罪（波罗夷）

大臣统理国土依倚王力，犯五根本罪：取兜婆物及四方僧物；破坏村邑城郭国土；毁谤正法舍二乘及无上乘；逼令持戒毁戒者还俗或加害；作五逆罪。犯一者善根烧然堕恶趣。
