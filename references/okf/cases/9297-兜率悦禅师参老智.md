# 兜率悦禅师参老智

兜率悦禅师在道吾首众，领数十衲子谒老智和尚，智与语未及数句，知悦所蕴深浅，笑曰观首座气质不凡，奈何出言吐气如醉人耶。悦面热汗下，愿和尚不吝慈悲，复与语未几，又锥札之，悦茫然。
