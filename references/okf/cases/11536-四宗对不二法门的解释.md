# 四宗对不二法门的解释

有人言不二法门即真谛理（成实论师）；有人言即实相般若（智度论师）；有人言即阿梨耶识（旧十地论师）；有人言即阿摩罗识（摄大乘论师）。初二约境，后两据心，同超四句。
