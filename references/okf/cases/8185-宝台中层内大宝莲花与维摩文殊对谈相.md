# 宝台中层内大宝莲花与维摩文殊对谈相

宝台中层内有大宝莲花，居七宝池。池中金龙衔花盘住花上。又有十三亿七宝楼，楼中有白玉宝塔，塔有维摩文殊对谈之相。塔表有大摩尼珠，至时出声遍至大千，音如天鼓又放光明，光中化佛说诸如来无生大慧。于六斋日莲花即开，有福者见。每受具戒时金龙吐水，直至戒坛化为香云雨。此诸功德是拘那含佛所造。
