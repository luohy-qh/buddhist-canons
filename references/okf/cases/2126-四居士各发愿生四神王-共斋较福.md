# 四居士各发愿生四神王，共斋较福

四居士分别供养四道人，受余食后各发愿生天帝释宫、海中作龙、金翅鸟中、人王作子，寿尽往生为四神王，后共至摩竭王后园八关斋，各言己福第一，王请智臣披陀类以四色缯悬空喻决义。
