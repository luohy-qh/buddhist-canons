# 凡夫修观灭重业获三昧

凡夫人修此观门，虽造五逆一阐提等极重恶业，皆悉消灭，应时便获五种三昧：一刹那三昧、二微尘三昧、三白缕三昧、四隐显三昧、五安住三昧。
