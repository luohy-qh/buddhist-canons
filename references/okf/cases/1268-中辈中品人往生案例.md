# 中辈中品人往生案例

中品中生者，一日一夜受持八戒或沙弥戒具足戒，威仪无缺，临终见阿弥陀佛与眷属持七宝莲华来迎，坐莲华往生，经七日莲华敷，闻法得须陀洹，半劫成阿罗汉。
