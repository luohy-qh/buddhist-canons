# 破戒比丘因咒脱地狱苦

一破戒比丘临终佩戴此咒，死后堕阿鼻地狱，但因咒力令地狱众生苦痛止息、火聚消灭，阎罗王遣狱卒探查，发现塔中尸上有咒，比丘罪障消除，生三十三天。
