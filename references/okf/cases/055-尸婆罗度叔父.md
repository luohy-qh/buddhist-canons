# 尸婆罗度叔父

尸婆罗的叔父悭贪，先施梵志百千两金，后给尸婆罗臭秽恶食，但因福田良美，天叹为大施，叔父遂以百千两金与尸婆罗，尸婆罗转施同学，为叔父说法得道迹。
