# 婆娑须拔陀罗偷学毗婆沙

阿緰阇国法师婆娑须拔陀罗托为狂痴入罽宾，听毗婆沙十二年，诵持在心。四次欲出，前三次被夜叉执回，第四次被放。回国后宣示所学，令学徒速取。
