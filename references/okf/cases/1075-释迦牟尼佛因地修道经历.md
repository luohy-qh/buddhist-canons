# 释迦牟尼佛因地修道经历

释迦牟尼佛往昔于三大阿僧祇劫修学有为功德多诸过患，后遇大觉世尊为其略说法要，如说修行，在毗耶离城中入大等道场经七七四十九日遂证无生法忍，众善具足，百福庄严，万行圆备，遂成佛道。
