# 天帝释与梵天化师子教人持戒生天

天帝释化师子，梵天化婆罗门，以师子索人啖为缘，教罪人持五戒念十善，罪人还国后国人皆从受戒，死者生天。佛言释即自身，梵天即迦叶。
