# 对暴恶犯戒者弃舍不作饶益之犯戒

菩萨若因嫌恨、恚恼而弃舍暴恶犯戒有情不作饶益，是染违犯；若因懒堕懈怠弃舍，是非染违犯；若因狂乱、欲调伏彼、护僧制等，则无违犯。
