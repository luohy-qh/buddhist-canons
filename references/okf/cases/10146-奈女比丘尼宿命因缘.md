# 奈女比丘尼宿命因缘

奈女比丘尼过去世为贫女人，以一枚奈和一盂水奉迦叶佛及众僧，缘此福祚寿尽生天得为天后，下生世间不由胞胎，九十一劫生奈华中，端正鲜净。
