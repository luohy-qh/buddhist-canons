# 社会阶级的起源

以正法治民者称为刹利；惩罚偷盗者，重犯则造督遮、牢狱；厌世山林修梵行者称婆罗门；不乐禅坐者称不禅婆罗门；好营居业者称居士；好机巧者称首陀罗；厌世法服修道者称沙门。
