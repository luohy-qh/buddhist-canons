# 慧思遭恶论师毒害经历

年三十九时，有五人恶论师以生金药置饮食中令慧思食，慧思身怀极困得停七日气命垂尽，临死之际一心合掌向十方佛忏悔念般若波罗蜜，毒即消除还更得差。
