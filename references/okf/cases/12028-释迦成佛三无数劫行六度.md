# 释迦成佛三无数劫行六度

初无数劫奉侍七万五千佛（最后宝髻佛），第二无数劫奉侍七万六千佛（最后然灯佛），第三无数劫奉侍七万七千佛（最后胜观佛）。施度圆满于为普度太子时，忍戒圆满于为忍辱仙时，进波罗蜜圆满于赞底沙佛超九劫，定慧圆满于金刚座后夜金刚喻定。
