# 第三十回僧物自入戒

知施主许欲布施现前僧物而心未定，回求入己即犯此罪。若回四方僧物入现前准律突吉罗，以不入己故。若入己者亦应犯舍。若知决定是施僧心定回入己者理即犯重。
