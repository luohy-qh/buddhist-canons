# 宝楼中层弥勒菩萨观

宝楼中层内有十六介七宝楼观，随观有多弥勒菩萨像。经中善财童子遇弥勒菩萨于斯观中。又五百白银台台内各有五百金台，随台各有七宝池，各有千叶金莲花，随花有佛。复有七宝莲花小小，诸天童子等作天伎乐中有碧须弥山，山上作释迦如来度诸龙像。上来宝楼并是无言菩萨于维卫佛时造而供养。
