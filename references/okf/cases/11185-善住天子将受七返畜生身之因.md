# 善住天子将受七返畜生身之因

大智度论、提谓经、法华经等经论说明不同畜生身的业因：猪身因邪憍慢、痴虚受供、谤经、饮酒造痴业、恶口悭贪；野干身因轻慢善人、奸猾语；猕猴身因轻躁短促、游戏放逸；蟒蛇身因嗔恚多。
