# 影响众

如舍那品中。十方各十亿佛土微尘数等大菩萨来。一一各将一佛世界尘数菩萨。以为眷属。一一菩萨各兴一佛世界微尘数等妙庄严云。悉皆弥覆充满虚空。随所来方结跏趺坐。次第坐已。一切毛孔各出十佛世界微尘数等一切妙宝净光明云。一一光中。各出十佛世界尘数菩萨一切法界方便海。充满一切微尘道。一一尘中有十佛世界尘数佛刹。一一刹中三世诸佛。皆悉显现。念念中于一一世界。各化一佛刹尘数众生。
