# 睒摩迦孝养盲父母

睒摩迦在山中取好果、鲜花、美水供养年老盲目的父母。迦尸国王游猎时误以毒箭射中他。睒摩迦不计自身痛苦，只忧父母无人供养。后发实誓，毒箭自出，身体平复。
