# 博戏喻输赢与陆亘南泉双陆

法演以博戏喻此事，输赢皆在，并举陆亘与南泉看双陆，陆亘问‘恁么不恁么便恁么信彩去时如何’，南泉答‘臭骨头十八’，法演说外人闻得便来捉赌时，白云自有道理，引龙牙偈‘学道先须有悟由，还如曾斗快龙舟。虽然旧阁闲田地，一度赢来方始休。’
