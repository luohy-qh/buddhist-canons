# 错误事佛者的行为

不值明师，无经像，不礼敬，强教人受法戒，无至信，犯众戒，心意蒙冥，犹豫，不读经行道，乍信乍不信，不斋日烧香燃灯作礼，嗔恚骂詈，咒咀，怀憎嫉，使人杀生，不敬经像。
