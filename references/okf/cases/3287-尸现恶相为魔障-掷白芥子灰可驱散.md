# 尸现恶相为魔障，掷白芥子灰可驱散

持诵时若尸立现诸恶相，即知是魔障，行人可取白芥子和灰，诵佛顶王真言掷彼尸面，以真言大威力故，诸作障者驰散四方，障魔去已尸卧如初。
