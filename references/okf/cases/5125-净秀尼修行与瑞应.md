# 净秀尼修行与瑞应

净秀本姓梁，安定乌氏人。七岁自然持斋，闻断鱼肉即便蔬食。十二求出家，十九方得听许。事师竭诚，三业勤修。曾因欲讲十诵律但有钱一千，夜梦见鸦鹊等助讲，后七十檀越争设妙供。又曾七日供养礼忏，见二胡僧举手共语。齐文惠帝、竟陵文宣王厚相礼待。梁天监五年卒，年八十九。
