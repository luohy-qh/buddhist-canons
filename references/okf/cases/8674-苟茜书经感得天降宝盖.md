# 苟茜书经感得天降宝盖

绵州巴西县得第二果的苟茜，因被村人逼迫，发愿在草中仰卧向空书写《金刚般若经》，七日完成，诸天于上造宝盖覆之，地遂无草。村人因此信敬，设栏保护，斋日设会。
