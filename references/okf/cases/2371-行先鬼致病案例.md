# 行先鬼致病案例

行先鬼在特定日子行病，使人失精神如痴，一年内不祭则必死，属危宿。需用黄纸钱一百二十贯、好酒脯祭之，当日精神平复，鬼永不敢到门，且需钉其身上下。
