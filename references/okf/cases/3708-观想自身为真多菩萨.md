# 观想自身为真多菩萨

行者变宝莲为真多菩萨，手持如意宝，六臂身金色，顶髻宝庄严，冠坐自在王，住说法相：第一手思惟，第二持意宝，第三持念珠，左按光明山，第二持莲手，第三挈轮手。
