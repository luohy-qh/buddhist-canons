# 日照商主请僧供养及金色苾刍助施

日照商主延请阿难及诸苾刍众，备饮食及价值百千上妙之衣奉施。金色苾刍知其父意，脱自身所被金色三衣次第奉施三俱胝众，旋脱旋生衣无竭尽。
