# 色之患证：老病死变

见女人年老（八十至百二十岁）头白齿落、面皱皮缓、身重少气、拄杖偻行；死后一日至五六日身色变青、膀胀烂臭、虫食其肉；乃至皮肉离体但见白骨、骨节解碎。皆证前时端正姝好之没不现。
