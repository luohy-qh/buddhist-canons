# 武王分封诸侯

武王追思先圣，褒封神农之后于焦，黄帝之后于祝，帝尧之后于苏，帝舜之后于陈，大禹之后于杞；封功臣谋士，师尚父于营丘曰齐，周公于曲阜曰鲁，召公于蓟曰北燕等，制天下立七十一国，封兄弟之国十有五人，姬姓之国四十人。
