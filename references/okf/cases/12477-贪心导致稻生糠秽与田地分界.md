# 贪心导致稻生糠秽与田地分界

有情因慵懒，旦起取稻兼两时粮，同伴效仿，渐取多日粮。贪心日增，稻中生糠秽，一刈后不再生，实渐小恶。诸人共集，悲叹后，分取地界，封量疆界，各各分之，世间田地始为耕种，遂立疆畔。
