# 九种戒差别

凡夫戒入生处故尽，外道五通戒神通退故尽，人戒十善业道尽故尽，欲界天子戒福尽故尽，色界天子戒禅那尽故尽，无色界天子戒三摩钵帝尽故尽，学无学声闻戒究竟涅槃故尽，独觉戒阙大悲故尽，菩萨戒无有尽。
