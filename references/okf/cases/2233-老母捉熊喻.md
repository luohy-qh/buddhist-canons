# 老母捉熊喻

老母绕树避熊，熊一手抱树，老母合树捺熊两手，唤人共捉，既捉之后老母舍熊走，其人被熊困。比喻凡夫作诸异论不善好，文辞繁重，竟不成讫便舍终亡，后人捉之欲解释，反为其困。
