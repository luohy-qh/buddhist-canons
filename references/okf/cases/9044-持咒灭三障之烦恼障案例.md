# 持咒灭三障之烦恼障案例

断除三毒根成佛道无疑。大火从四面来焚烧己身，龙王降雨。设火焚身节节疼痛，三诵此咒即得除愈。不被系缚。长贪欲嗔恚愚痴三毒等畏，譬如猛风吹去重云，即得消伏。
