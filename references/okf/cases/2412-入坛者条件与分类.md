# 入坛者条件与分类

入坛者若王若臣若诸凡夫，想持愿者与佛平等，能益自他、不生恶趣。坛法分三种：地坛（为王）、国坛（为大臣）、民坛（为凡人），大小依次为大、中、小，若不依法便生恶事。
