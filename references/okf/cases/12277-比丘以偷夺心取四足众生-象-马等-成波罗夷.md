# 比丘以偷夺心取四足众生（象、马等）成波罗夷

比丘以偷夺心解绳牵四足众生（象、马、牛、羊等）过四踔，或驱出墙障过四踔，即波罗夷；杀已取肉直五钱亦波罗夷。
