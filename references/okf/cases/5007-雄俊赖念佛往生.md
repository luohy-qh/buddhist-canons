# 雄俊赖念佛往生

唐雄俊不守戒律，闻‘一称佛名灭八十亿劫生死重罪’大喜，虽为恶而念佛不辍，暴亡经宿苏，言至冥司主者云误追，令还世倍精进，后斋戒念佛四年，端坐而化，言‘以念佛得生净土’。
