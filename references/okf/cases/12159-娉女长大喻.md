# 娉女长大喻

人以钱娉求人家小女，后女长大，他人复求娶为妇。前娉家争辩，后家称自娶大时妇。王判前娉家直，因女本小今长大，是前娉家妇。喻今世用名身作善恶，乃生后世，今世作善恶是其本。
