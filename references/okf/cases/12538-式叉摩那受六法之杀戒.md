# 式叉摩那受六法之杀戒

式叉摩那若自手断人命，或持刀授人、教死、赞死、劝死、与非药、堕胎、厌祷、咒术，自做或教人做，则非式叉摩那非释种女；断畜生不能变化者命缺戒，应更与受戒。
