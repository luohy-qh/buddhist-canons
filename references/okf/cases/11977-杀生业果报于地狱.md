# 杀生业果报于地狱

地狱中罪人，狱卒教示其自造恶业决定无失。罪人受罪乃至地狱果满，见如大黑云、金刚嘴大恶狗众，狗来搏噬，皮肉丝裂、筋脉断坏、支体骨节各各分离，食啖无遗，业力所持而复还生，展转长时。此由杀生取其肉食，害有情故。
