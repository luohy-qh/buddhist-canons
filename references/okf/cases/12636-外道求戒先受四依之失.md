# 外道求戒先受四依之失

有一年少外道来求出家，诸比丘先与四依，外道报言只堪受二依（乞食、树下坐），不堪纳衣、腐烂药，遂不出家。佛言若出家当得道证，自今已去先受戒已后受四依。
