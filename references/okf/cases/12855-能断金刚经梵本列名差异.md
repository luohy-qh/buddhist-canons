# 能断金刚经梵本列名差异

依杜行顗梵本，能断金刚经最初列八名（有情、命者、士夫、数取趣、意生、摩纳婆、作者、受者），后文说九（加‘我’）。大般若等处则说十三、十五、十七、十九等不同数目，因开合单重通别行相不同。
