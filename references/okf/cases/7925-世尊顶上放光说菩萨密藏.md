# 世尊顶上放光说菩萨密藏

世尊从顶上放百千种光明，光明化出莲华台座，座上化佛说菩萨密藏修罗及陀罗尼金刚章句，清净三轮如来诸力无畏法，能净无量众生令住不退转地。
