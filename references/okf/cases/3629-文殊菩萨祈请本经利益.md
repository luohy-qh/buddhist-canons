# 文殊菩萨祈请本经利益

文殊菩萨怜愍恶世众生，祈请世尊，若有听闻此经者，于无上菩提不退转，诸罪消灭，罪魔不侵，一切灾害不如意事消散，所求顺缘随意成就。并说若有持诵、书写、供养此经者，四大天王等天众皆诣其所，无复横死，精气不夺。
