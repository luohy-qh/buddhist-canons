# 国王求丰实免怨贼（娑伽罗龙王）

国王欲得丰实、无他怨贼，应炽然正法、恩惠普覆、赈诸贫穷、怜愍孤老，并于净洁处七日中不食酒肉五辛，白净素食，香汤沐浴，著白净衣，受持八戒，烧香散华，诵咒二七遍，三称龙王名。
