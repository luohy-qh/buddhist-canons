# 菩萨纳妻耶输陀罗为满宿愿

菩萨纳耶输陀罗为妻，是因为耶输陀罗往昔于燃灯佛所发愿，要在释迦牟尼佛法中为释种女种善根，菩萨为令其满宿愿而纳为妻，并非出于贪爱。
