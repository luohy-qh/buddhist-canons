# 东南方火天形象与印真言

火天乘青羊，赤肉色遍身火焰，右二手一持青竹一持军持，左二手一杨掌一持念珠。有二天女持天花，左右置苦行仙。印相：左手作拳安腰左，右手五指直竖相著，屈空纳著掌中，风屈中节，与娑婆诃相招。真言：南莫三满多没驮南．阿哦那(二合)曳娑嚩(二合引)诃(引)。
